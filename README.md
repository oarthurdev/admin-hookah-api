# Admin Panel API

Admin Panel API for Priston Tale servers.

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install this project.

```bash
npm install
```

## Structure and fill the database.

```zsh
npx knex migrate:latest
npx knex seed:run
```

## Start API

```zsh
npm start
```
